The Best Practices

Saturday, February 27 2016

Classes and Id's are two important selectors in CSS because they allow you to CLASSify and IDentify specific parts of your site for design. Without them, websites would be extremely simple and one dimensional.

The class selector, is useful because the same class can be applied to multiple elements. This can save a lot of time because the CSS doesn't have to be rewritten for different areas of your website. You can simply apply the same class to the element and the CSS will be the same.

To select a class in CSS: .CLASSNAME {}

The ID selector is different from the class selector because each ID can only be used once in a document. Therefore, it is good for usage on parts of your site that need specific design qualities.

To select an ID in CSS: #IDNAME {}
