JavaScript Cheat Sheet - The Basics of Objects

In Javascript, the way to organize and store data is with objects. JavaScript objects are very similar to Ruby hashes because they are set up in the same key,value format. For objects, the "keys" are called properties and describe the object, they are basically the same as normal variables except they are exclusively associated to the object it's in. Usually objects are described by comparing them to real life things. For example, the cup was the object, the cup has properties such as size, color, and brand.

Adding and deleting properties

To add and delete properties to an object is very simple. To add a property, use the same syntax as you would if you were trying to access a property and then set it equal to what you want. If it does not exist, it will automatically add the property, if it does, it will update the property.

To delete a property simply type delete and then the same syntax you would use to add the property.

Defining a method/functions inside an object

You can define a function inside of an object that can be used on the object itself. They are defined just like normal functions excpet you have to assign them as a value of a property.

Now you can call that function on the object, using the method name, which is the property name.
